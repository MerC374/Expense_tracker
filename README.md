# Expense Tracker 💸

A full-stack web app to track daily expenses with AI-powered insights.

## Features
- Add and categorize daily expenses
- Weekly pie chart visualization
- AI insights powered by LLaMA3 (Groq API)
- Weekly spending summary
- Secure API key management with environment variables

## Tech Stack
Python, Flask, SQLite, Chart.js, Groq API, LLaMA3

## Setup
pip install flask groq python-dotenv
Create a .env file with your GROQ_API_KEY
python app.py
